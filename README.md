#telkom-internship-qa-testing
QA Testing documentation from internship at **Telkom Indonesia** — PADI UMKM e-commerce platform

## Description
This repository contains documentation for the 130 test cases created during an internship program as a QA Engineer at Telkom Indonesia, focusing on quality assurance for PADI UMKM platform — an e-commerce platform dedicated to small and medium enterprises (SMEs) in Indonesia.

## Tools Used

- **Manual Testing** — Functional testing directly through the application interface
- **Postman** — API endpoint testing for response validation, status codes, and data integrity

## Testing Methodology

- **Functional Testing** — Ensuring every feature works according to requirements
- **API Testing** — Validating endpoints using Postman (request/response, status codes, authentication)
- **Positive & Negative Testing** — Testing both valid and invalid scenarios
- **Regression Testing** — Ensuring existing features are not affected after changes
